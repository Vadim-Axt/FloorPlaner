# FloorPlaner

Редактор планов квартир и домов в браузере. Рисуйте стены, расставляйте двери, окна и мебель, переключайте этажи и сохраняйте проект в файл.

## Запуск

```bash
npm install
npm run dev
```

Сборка для публикации:

```bash
npm run build
```

## Возможности

- **Квартира или дом** — для дома доступно до 5 этажей, каждый этаж отдельный слой.
- **Инструменты** — стены (линия по двум точкам), выбор, перемещение холста.
- **Объекты** — дверь, окно, комната, кровать, диван, стол с типовыми размерами в сантиметрах.
- **Свойства** — у выбранного объекта можно изменить ширину, длину и поворот.
- **История** — отмена и возврат действий.
- **Буфер** — копирование и вставка объектов с просмотром содержимого буфера.
- **Сохранение** — экспорт плана в JSON и открытие ранее сохранённого файла.
- **Темы** — светлая и мягкая (тёплые оттенки).

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `1` | Стена |
| `2` | Выбор |
| `3` | Рука |
| `4`–`9` | Объекты (дверь, окно, комната, кровать, диван, стол) |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Вернуть |
| `Ctrl+S` | Сохранить |
| `Ctrl+C` | Копировать |
| `Ctrl+V` | Вставить |
| `Delete` | Удалить выбранное |
| Колёсико мыши | Масштаб холста |

## Масштаб плана

Сетка на холсте шагом **20 см**. Один пиксель на холсте соответствует одному сантиметру. Сетка закреплена на экране и не двигается при панорамировании.

## Формат файла

Проект сохраняется в JSON. В файле хранятся тип здания, число этажей и список объектов с координатами и размерами.
